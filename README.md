# Joel Tsewagbe - Portfolio Website

A modern, responsive portfolio website showcasing Joel's skills as a Digital Marketer and Analytical Expert with a strong foundation in Computer Science.

## 🌟 Features

- **Modern Design**: Clean, professional layout with smooth animations
- **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic navigation
- **Performance Optimized**: Fast loading with optimized assets
- **SEO Friendly**: Proper meta tags and semantic HTML structure
- **Accessibility**: WCAG compliant with proper contrast and keyboard navigation

## 🚀 Quick Start

1. **Download/Clone** the files to your local machine
2. **Open** `index.html` in your web browser
3. **Customize** the content to match your information
4. **Deploy** to your preferred hosting service

## 🎨 Customization Guide

### Personal Information
Edit the following sections in `index.html`:

- **Name**: Update "Joel Tsewagbe" throughout the file
- **Email**: Replace `[your email]` with your actual email address
- **LinkedIn**: Update the LinkedIn URL if needed
- **Location**: Change "Ghana" to your location

### Content Updates

#### Hero Section
- Update the main title and subtitle
- Modify the tagline: *"Turning data into decisions, and ideas into impact."*
- Change the call-to-action buttons

#### About Section
- Update the personal description
- Modify your experience at SlimFile
- Adjust your goals and aspirations

#### Experience Section
- Update job title, company, and dates
- Modify responsibilities and achievements
- Add more experience entries if needed

#### Education Section
- Update university names and degree information
- Change status from "Ongoing" to "Completed" when finished
- Add graduation dates when applicable

#### Skills Section
- Add or remove skills as needed
- Update skill categories and tags
- Modify icons for different skill types

#### Projects Section
- Update project descriptions
- Add links to live projects if available
- Modify project tags and technologies used

### Styling Customization

#### Colors
The website uses a modern color scheme. To change colors, update these CSS variables in `styles.css`:

```css
/* Primary Colors */
--primary-blue: #2563eb;
--primary-purple: #7c3aed;
--accent-yellow:rgb(19, 18, 16);

/* Background Colors */
--bg-light: #f8fafc;
--bg-dark: #1f2937;
```

#### Fonts
The website uses Inter font family. To change fonts:

1. Update the Google Fonts link in `index.html`
2. Modify the `font-family` property in `styles.css`

## 🌐 Deployment Options

### GitHub Pages
1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select source branch (usually `main`)
5. Your site will be available at `https://username.github.io/repository-name`

### Netlify
1. Drag and drop the folder to [Netlify](https://netlify.com)
2. Your site will be deployed instantly
3. Customize the domain name in settings

### Vercel
1. Connect your GitHub repository to [Vercel](https://vercel.com)
2. Deploy automatically on every push
3. Get a custom domain and SSL certificate
